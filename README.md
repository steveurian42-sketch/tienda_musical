# En Clave de Fa — Tienda de Instrumentos

Pequeña tienda de venta de instrumentos y accesorios creada como trabajo práctico de Front-End.

Objetivo
- Implementar una página web con estructura semántica, estilos responsivos, formulario de contacto y un carrito de compras en JavaScript.

Archivos relevantes
- `pages/index.html` — Página principal con listado de productos (se inyectan con JS).
- `pages/contacto.html` — Formulario de contacto (Envía con Formspree).
- `pages/carrito.html` — Vista del carrito de compras.
- `css/style.css` — Estilos principales.
- `js/` — Lógica de interacción (productos, carrito, storage, UI).

Instrucciones rápidas
1. Abrir `pages/index.html` en el navegador (puedes usar Live Server o abrir el archivo directamente).
2. Para probar el formulario de contacto, usar `pages/contacto.html`.
